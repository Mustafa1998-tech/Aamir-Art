# Aamir Art — Furniture Designer & Creative Artisan

Premium minimalist portfolio website for Aamir Art, featuring handcrafted furniture and decor projects.

## Structure

```
index.html       — Main page with all project cards, modal, contact
styles.css       — Design system with CSS variables, responsive breakpoints
script.js        — Interactions: scroll reveal, smooth scroll, modal, counters
assets/images/   — Project folders (image-1.jpg, image-N.jpg, video.mp4)
```

Each project folder under `assets/images/` contains:
- `image-1.jpg` through `image-N.jpg`
- `video.mp4` (optional, autoplay loop muted)

## Features

- Single-column full-width project sections with 4-column image grid
- First image spans 2×2 for hero treatment
- Video autoplay loop muted (no controls)
- Scroll-triggered reveal animations
- Stat counter animation on scroll
- Scroll progress bar
- Project modal gallery with zoom on hover
- Responsive: mobile, tablet, desktop
